this 的四种指向

  1. 第一种：

    function foo(){
    console.log(this);
}
foo()
    • 当调用 foo() 的时候, this 的指向是 window对象。

  2. 第二种：

    var obj = {
    sayHi: function() {
        console.log("hello, girl.");
        console.log(this === obj);
    }
};
obj.sayHi();
    • 结果为：hello, girl. 和 true
    • 这个时候 this -> 该方法的调用者
    • 也就是 对象 obj

  3. 第三种：

    function fn(name, age, gender) {
    this.name = name;
    this.age = age;
     console.log(this);
}
var f = new fn('f');
    • 结果为：fn {name: "f", age: 18}
    • 这个时候 this -> 当前要创建出来的对象（或 实例）
    • 也就是对象 对象 f

    • 可以将其理解为：（仅做参考）

      f = {
    'name':'f',
    'age':18
}

  4. 第四种： call / apply

    这种模式可以用在（上下文模式）而且只有函数才有 call和 apply 方法。

    • call 模式

      1. fn.call( thisObj, arg1, arg2, ...,argN )

        • thisObj: 是 fn 在执行时的 this 指向
        • arg1, arg2, ...,argN： 是 fn 在执行时传入的实参列表

        • 动态指定某方法或函数执行时，内部的 this 的指向

          function fn(name, age, gender) {
    this.name = name;
    this.age = age;
    this.gender = gender;
     console.log(this);
}
var o = {};
fn.call(o, 'o', 18, '男');
        • 结果为： Object {name: "o", age: 18, gender: "man"}

    • apply 模式

      1. <fn>.apply( thisObj, [fn实参])

        • thisObj: 是 fn 在执行时的 this 指向
        • [fn实参]: 传入 数组 或 伪数组对象 实参

        • 也是 动态指定某方法或函数执行时，内部的 this 的指向

          function fn(name, age, gender) {
    this.name = name;
    this.age = age;
    this.gender = gender;
     console.log(this);
}
var o = {};
fn.apply( o, ['o', 18, 'man'] );
        • 结果为：Object {name: "o", age: 18, gender: "man"}

        • 举一个例子：

          var obj = {num1:1, num2: 2, num3: 3};
function foo(num1,num2,num3){
    console.log(num1 + num2 + num3);
}
var obj1 = { 0:1, 1:2, 2:3 ,length:3};
foo.apply(obj, obj1);
        • 结果为： 6
        • 所以它可以传入数组或伪数组对象，它会将上面的元素当做对象传入到 foo 的实参里面，且它不论参数是数组或伪数组，都只会将 自然数 传入到实参里面，而不会将 length 传入，这是它的一个特点。

this 的指向： this 的指向就是指向它的调用者，谁调用这个函数，this 就指向谁。
